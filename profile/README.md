## 🚀 私たちのミッション

当組織（Junior IT Certification Association）は、次世代を担うプレイヤーのためのITリテラシー向上と、高度なIT技術への挑戦を支援するプラットフォームを提供しています。
「初心者向けの基礎」から「大人が頭を抱える超難関レベル」まで、遊び心と本気が詰まったクイズアプリを開発しています。

## 🏆 主なプロジェクトと構成

### 1. IT理解度確認テスト (quiz-repo)
HTML/JavaScriptで軽量に動作する、ブラウザ完結型のWebクイズアプリケーションです。

*   **🔥 小学生向けIT理解度確認テスト**
    *   **本シリーズにおける最高峰・最難関テスト**。未来のテックリーダーを育成するための超ハイレベルなITリテラシーと深い知識が要求される、当リポジトリ最大の挑戦状です。
*   **🥇 1級 / 準1級**
    *   最高難易度「小学生向け」への登竜門となる、応用・実践的なトレンド技術や高度なセキュリティ知識を問うクラス。
*   **🥈 2級 / 🥉 3級**
    *   日常や実務で役立つ踏み込んだリテラシーから、基礎的な用語を網羅したエントリークラス。

👉 **[IT理解度確認テストを実際に受検する（公開サイト）](https://jr-it.rweb.site)**

# Junior IT Certification Association (jr-it-ex)

<samp>Can’t even adults pass this quiz!? An ultra-authentic IT comprehension quiz project designed to nurture and evaluate future genius engineers and the next generation of IT leaders. </samp>

---

## 🚀 Our Mission

Our organization (Junior IT Certification Association) provides a platform to help the next generation of IT professionals improve their IT literacy and take on challenges in advanced IT technologies.
We develop quiz apps packed with both fun and seriousness, ranging from “basics for beginners” to “extremely difficult levels that even adults struggle with.”

## 🏆 Main Projects and Structure

### 1. IT Proficiency Test (quiz-repo)
A browser-based web quiz application that runs lightweight in HTML/JavaScript.

*   **🔥 IT Proficiency Test for Elementary School Students**
    *   **The pinnacle and most challenging test in this series**. This is the biggest challenge in our repository, requiring ultra-high-level IT literacy and deep knowledge to nurture future tech leaders.
*   **🥇 Level 1 / Pre-Level 1**
    *   A class that serves as the gateway to the highest difficulty level—“for elementary school students”—testing applied and practical knowledge of trending technologies and advanced security concepts.
*   **🥈 Level 2 / 🥉 Level 3**
    *   An entry-level class covering everything from in-depth literacy useful in daily life and work to basic terminology.

👉 **[Take the IT Proficiency Test (Public Site)](https://jr-it.rweb.site)**
